1. Introduction

Structs has similar concepts as in Schema.org. But it is more machine-readable than human-readable.

When comparing to Schema.org, it has the following differences.

  • it supplies .ini format more than description written on web pages
  • it keeps ECS (entity-components-system paradigm) in mind, but not as a hierarchy tree
  • tools integrated (coming soon)

Let's take the person as an example.
